CRYPTOCRACKER

An online Quizzing Portal made for my college technical cultural fest in order to conduct multiple choice type question based quiz. The Feature in the quizzing portal include the following:

  • Login as a contestant or as a creator of a quiz
  • As a creator of a quiz you can create multiple question banks. Can create multiple quizes by selecting a question from the question banks, and can schedule the quiz to be conducted at a specific time.
  • As a contestant you can register to participate in a particular quiz, can participate in any number of quiz (single or in teams)
  • Features are provided to prevent the contestant from cheating.
  • There is a live leaderboard that shows the contestant in order of their rank.

Technologies used are :

  • Python/Django
  • Django Rest
  • Mysql
  • AngularJs

SHRINKKER

Shrinkker is an online service to shorten the url. The url is converted into a uniqueid that is then converted into a 62 base number that ehich is stored in the database(Redis). One can add a url, the service will shorten the url and store the mapping. When given a short address the service will provide the corresponding full url.

Technologies used are :

  • Java
  • Spring Boot

ONLINE COURSE MANAGEMENT(FUSION IIIT)

OCMS is a module in the ERP of IIITDM Jabalpur (FUSION IIIT). OCMS module had the following features:

  • There are 2 users for this module, i.e., student and professor
  • The module manages all the courses of all the years for the respective semesters, for both the student and professor
  • As a professor I will be shown all the course I have under me in the current semester. I can upload course materials like video tutorials, pdf docs, etc. Can also upload assignmets for the students to submit. There is a option to download and grade all the submitted assigment. Professor can create any number of question banks for any course individually and can create and schedule any number of quizes by selecting questions from the question bank.
  • As a student one can see all the semester courses one is enrolled in. Can download all the course material upload by the professor. Can submit the assigments, and can take part in the quiz that is being conducted.

Technologies used are :

  • Python/Django
  • Django Rest
  • Mysql
  • AngularJs
